Projects

Helmet Detection using YOLOv8

This project aims to automatically detect whether motorcyclists are wearing helmets using the YOLOv8 object detection model. Images or video frames are analyzed to identify motorcyclists and classify them as wearing or not wearing helmets. The system is trained using a labeled dataset and deployed using Python and tools like Google Colab and OpenCV. It helps improve road safety and enables smart surveillance and traffic law enforcement.

GitHub Live Demo

PupilHeart: HRV Monitoring via Pupillary Fluctuations

PupilHeart introduces an innovative, non-invasive approach to monitoring Heart Rate Variability (HRV) by analyzing pupillary fluctuations using standard mobile device cameras. This groundbreaking method offers a convenient, contactless, and cost-effective alternative to traditional HRV measurement tools.
